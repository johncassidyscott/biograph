About BioGraph

BioGraph is a life sciences knowledge graph that connects drugs, diseases, targets, clinical trials, and companies. It integrates data from authoritative sources including MeSH, ChEMBL, ClinicalTrials.gov, and OpenTargets.

POC Coverage

  • Disease Areas: Obesity/Metabolic, Alzheimer's Disease, KRAS Oncology
  • Time Window: January 2024 - January 2025
  • Data Sources: MeSH, ChEMBL, ClinicalTrials.gov, OpenTargets

Entity Types

  • Drugs: Small molecules and biologics with ChEMBL IDs
  • Targets: Protein targets with UniProt IDs
  • Diseases: Conditions from MeSH ontology
  • Trials: Clinical studies from ClinicalTrials.gov
  • Companies: Pharmaceutical sponsors

API Documentation: Visit /docs for interactive API documentation.
